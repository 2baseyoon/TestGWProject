웹파트 도움말

웹파트 형태 : 웹파트 위치 설정을 할 수 있습니다.
이름 값 설명
고정 FIX 사용자의 선택없이 고정값 (위치변경은 가능하나 삭제 불가/ 포탈 초기화시 자동 선택)
기본 DEF 사용자의 선택없을 시 기본값(위치변경 삭제 가능/ 포탈 초기화시 자동 선택)
서비스 장치 : 설정 웹파트를 제공하는 서비스 장치를 나타냅니다.
이름 설명
PC/Mobile PC Mobile 모두 제공
PC PC에서만 제공
Mobile Mobile에서만 제공
웹파트 클래스명 : 해당 웹파트에 맞는 클래스를 입력합니다.
클래스명 설명
웹파트 크기 : 웹파트 크기를 설정할 수 있습니다.
위치 값 설명
x x1 : x영역 1만큼, x2 : x영역 2만큼 웹파트 x영역의 크기를 설정할 수 있습니다.
y y1 : y영역 1만큼, y2 : y영역 2만큼 웹파트 y영역의 크기를 설정할 수 있습니다.
웹파트 확장 JSON : 웹파트 크기를 설정할 수 있습니다.
Key Value 설명
moreUrl 해당 웹파트의 상세 정보가 있는 페이지 url 웹파트 우측 상단에 더보기 버튼과 링크를 생성합니다.
mobileMoreUrl 해당 웹파트의 상세 정보가 있는 페이지 url(모바일) 웹파트 우측 상단에 더보기 버튼과 링크를 생성합니다.(모바일)
popupFn 팝업창 실행 함수명 연결된 팝업창을 실행합니다.
swiperInfo {"slidesPerGroup": 한 슬라이드에 보여지는 데이터 수} 목록 갯수에 맞게 스와이프 처리를 합니다.
chngColor Y : 변경 가능, N : 변경 불가능 웹파트 색상 변경 가능 여부
