단축키

편집

실행 취소

  1. - 최근에 편집한 동작을 취소하고 이전 상태로 편집을 되돌립니다.
  2. - 실행 취소는 최근의 편집 동작 개별로 한 단계씩 취소됩니다.
  3. - 더 이상 되돌리기를 할 수 없는 경우에는 버튼이 비활성화 됩니다.

다시 실행

  1. - 실행 취소했던 작업을 다시 원래의 실행 상태로 되돌립니다.
  2. - 다시 실행은 최근의 실행 취소 동작을 한 단계씩 다시 실행합니다.
  3. - 더 이상 다시 실행할 수 없는 경우에는 버튼이 비활성화 됩니다.

잘라내기

  1. - 블록으로 선택되어 있는 부분이 클립보드로 복사되면서 사라집니다.
  2. - 잘라낸 부분이 클립보드에 보관되어 있으므로 다른 곳에 붙여 넣을 수 있습니다.
  3. - 클립보드에 새로운 내용이 복사되면 이전에 보관되어 있던 내용은 사라집니다.

복사

  1. - 블록으로 선택되어 있는 부분이 클립보드로 복사되며 선택된 부분은 그대로 남아있습니다.
  2. - 복사된 내용이 클립보드에 보관되어 있으므로 다른 곳에 붙여 넣을 수 있습니다.
  3. - 클립보드에 새로운 내용이 복사되면 이전에 클립보드에 보관되어 있던 내용은 사라집니다.

붙여넣기

  1. - 클립보드에 보관되어 있는 내용이 커서가 있는 위치로 삽입됩니다.
  2. - 붙여넣기를 하려면 먼저 [잘라 내기] 또는 [복사]등을 실행해야 됩니다.

서식복사

  1. - 블록으로 선택되어 있는 부분의 서식이 복사되며 선택된 부분의 서식은 그대로 남아있습니다.
  2. - 복사된 서식은 보관되어 있으므로 다른 곳에 붙여 넣을 수 있습니다.

서식 붙여넣기

  1. - 복사된 서식을 특정 블록에 적용할 수 있습니다.
  2. - 서식 붙여넣기를 하려면 먼저 서식 복사를 실행해야 합니다.

전체선택

  1. - 에디터 창에 있는 모든 내용이 블록으로 지정됩니다.
  2. - 이 상태에서 [잘라 내기], [복사], [지우기], [붙여넣기]등의 편집 작업을 할 수 있습니다.

찾기 및 바꾸기

현재 작성 중인 에디터에서 문자를 찾아서 교체하고 싶은 문자로 변경할 수 있습니다

  1. - 찾기 : 편집창에 찾고 싶은 문자를 입력합니다.
  2. - 교체 : 편집창에 교체하고 싶은 문자를 입력합니다.
  3. - 검색 옵션을 선택할 수 있습니다. (대소문자 일치 여부 등)
  4. - [찾기] 버튼을 선택하여 찾기를 실행합니다.
  5. - [교체]나 [전체 교체] 버튼을 선택하여 새로운 문자로 교체할 수 있습니다.
