웹 접근성 검증

총 5가지의 웹 접근성 검증 항목을 제공 합니다.

Table 태그

  1. - Table 태그의 Caption(제목)과 Summary(개요) 위반 여부를 검증합니다.
  2. - 위반 사항이 있으면 누락 항목 내용이 표시되며 누락 항목 내용을 입력할 수 있는 입력창이 나타납니다.

- 누락 항목 내용 입력 후 [적용] 버튼을 누르면 누락 항목 내용이 Table 태그에 적용됩니다.

Image 태그

  1. - Image 태그의 alt(설명) 위반 여부를 검증합니다.
  2. - 위반 사항이 있으면 누락 항목 내용이 표시되며 누락 항목 내용을 입력할 수 있는 입력창이 나타납니다.

- 누락 항목 내용 입력 후 [적용] 버튼을 누르면 누락 항목 내용이 Image 태그에 적용됩니다.

iframe 태그

  1. - iframe 태그의 title(제목) 위반 여부를 검증합니다.
  2. - 위반 사항이 있으면 누락 항목 내용이 표시되며 누락 항목 내용을 입력할 수 있는 입력창이 나타납니다.

- 누락 항목 내용 입력 후 [적용] 버튼을 누르면 누락 항목 내용이 iframe 태그에 적용됩니다.

A 태그

  1. - a 태그의 title(제목) 위반 여부를 검증합니다.
  2. - 위반 사항이 있으면 누락 항목 내용이 표시되며 누락 항목 내용을 입력할 수 있는 입력창이 나타납니다.

- 누락 항목 내용 입력 후 [적용] 버튼을 누르면 누락 항목 내용이 a 태그에 적용됩니다.

ID 중복 체크

  1. - 태그의 id 중복 위반 여부를 검증합니다.
  2. - 중복 위반 사항이 있으면 새로운 id를 입력할 수 있는 입력창이 나타납니다.

- 중복된 id를 새로운 id로 변경한 후 [적용] 버튼을 눌러서 id를 각각 변경합니다.

미입력 설정

- 미입력 체크박스를 체크 후 [적용]버튼을 누르면 위반 항목 여부를 체크하지 않습니다.
