제목 1 ⌃⌥1 Shift + Alt + 1
제목 2 ⌃⌥2 Shift + Alt + 2
제목 3 ⌃⌥3 Shift + Alt + 3
제목 4 ⌃⌥4 Shift + Alt + 4

툴바에 포커스 ⌥F10 Alt + F10
element path에 포커스 ⌥F11 Alt + F11

소문자로 변환 ⌘⇧L Ctrl + Shift + L
각 단어의 첫글자를 대문자로 변환 ⌘⇧C Ctrl + Shift + C
