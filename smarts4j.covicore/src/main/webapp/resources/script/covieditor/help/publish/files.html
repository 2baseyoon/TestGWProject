단축키

파일

새 문서

새로운 문서를 작성합니다.
기존에 편집 중인 내용은 화면에서 모두 제거됩니다.

파일 열기

선택한 파일의 내용을 편집기로 가져옵니다.(*지원 확장명: html, htm, txt)

  1. 파일 선택 후 [적용] 버튼을 누르면 “선택된 파일을 적용 하시겠습니까?” 팝업이 나타다고 [네] 버튼을 선택하면 파일의 내용이 에디터 편집창에 적용됩니다.

템플릿

서버에 저장되어 있는 문서 템플릿 양식을 사용자의 편집기로 로드하여 문서를 작성할 수 있습니다. (서버 템플릿 문서가 수정되는 것은 아닙니다)

템플릿 목록에서 사용자가 템플릿을 선택한 후 [적용] 버튼을 선택하면 템플릿의 내용이 에디터 편집창에 적용됩니다.

레이아웃

문서 레이아웃을 사용하여 문서작성을 할 수 있습니다.

사용하고 싶은 레이아웃을 선택하여 [적용] 버튼을 선택하면 레이아웃 내용이 에디터 화면에 적용됩니다.

저장

작성중인 내용이 사용자의 컴퓨터에 ‘Untitled.html’로 저장이 됩니다.

자동 저장 불러오기

환경 설정에서 ‘자동 저장 사용’이 설정되어 있으면 ‘자동 저장 불러오기’ 메뉴가 활성화 됩니다.

에디터에서 자동 저장되어 있는 html 내용을 에디터 화면에 불러올 수 있습니다.

최대 10개까지 최신 시간 순으로 저장이 됩니다. 저장된 html을 선택하여 [적용] 버튼을 선택하면 선택된 html 내용이 에디터 화면에 나타납니다.

인쇄

사용자의 컴퓨터에 설정되어 있는 프린터로 편집 화면의 내용을 인쇄합니다.
