단축키

도구

단어 수

  1. - 작성중인 문서의 단어 개수, 문자 개수를 확인할 수 있습니다.
  2. - 선택된 영역의 단어 개수, 문자 개수를 확인할 수 있습니다.

변환

  1. - 영문을 블록으로 선택하여 그 안에 있는 대문자와 소문자를 지정한 형식으로 바꿉니다.
  2. - 대문자로 변환 : 모든 글자를 대문자로 바꿔줍니다.
  3. - 소문자로 변환 : 모든 글자를 소문자로 바꿔줍니다.
  4. - 각 단어의 첫 글자를 대문자로 변환 : 각 단어의 첫 글자만 대문자로 바꿔줍니다.

개인정보 보호

  1. - 개인정보 유출 방지를 위해 주민등록번호, 전화번호, 이메일 패턴을 사용하여 개인정보 내용을 추출합니다
  2. - 추출된 개인정보 내용을 교체하기 위해서는 개인정보 리스트 중에 교체하고 싶은 내용을 선택한 후 교체 버튼을 사용하여 *표시로 교체할 수 있습니다.

환경 설정

- 에디터의 기본 사항을 설정할 수 있습니다.

  1. - 자동 저장 사용 : 작성 중인 문서를 자동 저장됩니다. (최대 10개)
  2. - 텍스트 선택 영역 도구모음 사용 : 텍스트 선택 영역의 퀵 메뉴가 보이도록 설정할 수 있습니다.
  3. - 삽입 도구모음 사용 : 퀵 삽입 메뉴가 보이도록 설정할 수 있습니다.
  4. - 기본 글꼴 : 기본 글꼴을 선택할 수 있습니다.
  5. - 기본 글자크기 : 기본 글자크기를 선택할 수 있습니다.
  6. - 기본 줄간격 : 기본 줄간격을 선택할 수 있습니다.
