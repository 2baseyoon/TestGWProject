단축키

삽입

배경 이미지

에디터 문서의 배경 이미지를 설정하거나 삭제합니다.

  1. - 소스 : 배경에 넣을 이미지를 선택할 수 있습니다.
  2. - 옵션 : 기본, 스크롤, 고정을 설정할 수 있습니다.
  3. - 반복 : 이미지의 반복 형식을 설정할 수 있습니다.
  4. - 배경색 : 배경색을 설정할 수 있습니다.
  5. - 배경 제거 : 체크박스를 체크하고 [저장] 버튼을 클릭하면 설정되었던 배경 이미지와 색이 모두 제거 됩니다.

이미지

에디터 문서에 이미지를 추가합니다.

  1. - [일반] 소스 : 배경에 넣을 이미지를 선택할 수 있습니다.
  2. - [일반] 대체 설명 : 이미지에 대한 대체 설명(Alt 속성) 속성을 편집할 수 있습니다.
  3. - [일반] 이미지 제목 : 이미지에 대한 제목(Title 속성) 속성을 편집할 수 있습니다. 별도로 편집하지 않을 경우 불러온 파일명이 기본으로 설정됩니다.
  4. - [일반] 캡션 표시 : 캡션 표시 체크 시 이미지 하단에 캡션을 넣을 수 있습니다.
  1. - [고급] 스타일 : 이미지의 스타일(이미지의 넓이, 높이 지정)을 직접 코드로 수정할 수 있습니다. (예: width:500px;height:600px; )
  2. - [고급] 수직공백 / 수평공백 : 이미지 스타일의 여백(margin)을 설정할 수 있습니다.
  3. - [고급] 테두리 두께 / 테두리 스타일 : 이미지 스타일의 border-style을 설정할 수 있습니다.

- [업로드] 이미지 끌어오기 / 이미지 찾기 : 이미지 찾기 또는 파일 드래그로 이미지를 삽입할 수 있습니다.

파일

파일을 삽입합니다.

  1. - 파일 버튼으로 추가할 파일을 선택할 수 있습니다.
  2. - 제목 : 입력한 문구는 삽입된 내용에 마우스를 올려놓으면 보이게 되는 문구입니다.

링크

하이퍼링크를 생성할 수 있습니다.

  1. - 주소 : 링크할 주소를 입력합니다.
  2. - 표시할 텍스트 : 에디터에 표시할 문구를 입력합니다.
  3. - 제목 : 입력한 문구는 삽입된 내용에 마우스를 올려놓으면 보이는 문구입니다.

코드 샘플

코드를 작성하고 작성한 텍스트를 에디터에 그대로 출력할 수 있습니다.

  1. - 언어 항목에서 작성할 코드 형식을 선택할 수 있습니다.
  2. - Code view : 삽입할 텍스트를 작성합니다.
  3. - [저장] 버튼을 선택하면 에디터에 Code view 내용이 표시됩니다.

특수 문자

에디터에 특수문자를 삽입합니다.

  1. - 검색창에 특수문자를 검색할 수 있습니다.
  2. - 특수문자를 클릭하면 팝업창이 닫히면서 선택한 특수문자가 편집창의 현재 커서 위치에 삽입됩니다

이모티콘

에디터에 이모티콘을 삽입합니다.

  1. - 검색창에 이모티콘을 검색할 수 있습니다.
  2. - 이모티콘을 클릭하면 팝업창이 닫히면서 선택한 특수문자가 에디터에 삽입됩니다.

가로

- 에디터에 수평선을 삽입합니다.

페이지 구분자

- 편집 내용 중간에 페이지 구분자를 삽입합니다. 페이지 구분자는 인쇄 시 페이지를 나눠서 출력할 수 있게 합니다.

책갈피

- 에디터에서 원하는 편집 위치로 이동하기 위한 책갈피를 삽입합니다 실제 HTML 소스에는 anchor tag을 삽입합니다.

  1. - 이름 : 책갈피 이름(ID)이며 입력된 이름은 <a> 태그의 id 속성이 됩니다. 따라서 반드시 영문자로 시작해야 합니다.
  2. - 추가 버튼을 누르면 현재의 커서 위치에 새로운 책갈피가 등록됩니다.
  3. - 기존에 있던 책갈피를 수정/삭제하기 위해서는 책갈피 목록 중 원하는 책갈피를 선택한 후 편집/삭제 버튼을 클릭하여 수정/삭제합니다.

날짜/시간

- 현재 시간이나 현재 날짜를 삽입할 수 있습니다. 날짜는 4가지의 포맷이 지원됩니다.
